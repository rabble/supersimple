## Usage in Code

These images are used in the marketing pages with the following paths:

```jsx
// For the hero section
<div 
  className="..."
  style={{
    backgroundImage: 'url(/images/unsplash/directory-showcase.jpg)',
    backgroundSize: 'cover',
    backgroundPosition: 'center'
  }}
>
```
